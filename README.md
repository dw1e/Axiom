# Axiom Anti-Cheat

## 简介
一个随便搞着玩玩的私人反作弊, 可能有一些没料到的误判/绕过, 看心情修~

由于本人不玩也不开高版本服务器, 所以也懒得支持高版本~

也就是说这些检测都是给 1.8 客户端设计的, 直接用在高版本服/玩家 = 误判++

懒得写太多注释, 所以不太推荐零基础小白看这个源码~

可以看作是 [FairFight](https://github.com/dw1e/FairFight) 重构

## 安装
**支持的版本:** 1.8 服务器上的 1.7 与 1.8 玩家 (不支持 高版本 与 跨版本 使用)

**硬♂依赖:** [ProtocolLib](https://www.spigotmc.org/resources/protocollib.1997/) 版本 5.0.0 及以上

**推荐使用的服务器:** 1.8 PVP 服
